Variables¶

  • \(\eta\) - Incident wave
  • \(\omega\) - Wave frequency (rad/s)
  • \(f\) - Wave frequency (Hz)
  • \(\theta\) - Wave direction (Degrees)
  • \(k\) - Wave number
  • \(\phi\) - Wave phase (rad)
  • \(H\) - Wave height (m)
  • \(H_{s}\) - Significant wave height, mean wave height of the tallest third of waves (m)
  • \(H_{m0}\) - Spectrally derived significant wave height (m)
  • \(A\) - Wave amplitude (m), half of the wave height (m)
  • \(m0\) - Spectral moment
  • \(T_{p}\) - Peak period (s)
  • \(T_{e}\) - Energy period* (s)
  • \(F_{exc}{\omega}\) - Wave excitation
  • \(A{\omega}\) - Added mass
  • \(B{\omega}\) - Radiation damping
  • \(K_{hs}\) - Hydrostatic stiffness
  • \(\rho\) - Water density
  • \(g\) - Gravity
